# Modernization baseline — captured 2026-07-02

Known-good state of JS9 v3.9.0 on this machine **before** any tooling changes.
Used as the reference point for the tooling-only modernization effort.

## Environment
- Node: v22.22.1 (the version installed here; forward target is Node 24 LTS)
- OS: Linux
- Java: **not installed** (see build note below)

## What works today
| Check | Result |
|-------|--------|
| `./configure` | ✅ exit 0, generates `Makefile`, `src/Makefile`, `astroem/jsfitsio/Makefile`, `src/js9helper.pc` |
| `make allinone` | ✅ exit 0, regenerates `js9-allinone.js` **byte-identical** (md5 `07d8675111235effdac35f7836d26c22`) — concat step is reproducible |
| Syntax check (`node --check`) | ✅ all pass: `js9.js`, `js9.min.js`, `js9-allinone.js`, `js9support.js`, `js9plugins.js`, `js9Helper.js`, `js9worker.js`, `astroem.js` |

## Known gaps / modernization pain points found
1. **Minification requires Java + Closure Compiler.** `js9.min.js` / `js9support.min.js` are
   regenerated by `build/minify` via `build/closure-compiler/compiler.jar`
   (**closure-compiler-v20190909.jar**, 2019-era) which needs `java`. Java is not installed here,
   so a *full* rebuild of the min files cannot run as-is. `make allinone` only worked because it
   concatenates the already-committed `.min.js` files.
   → A Vite/esbuild migration removes the JVM dependency entirely (JS-native minifier).
2. **Build is autotools + shell concat** (`configure.ac`, `Makefile.in`, `build/mkallinone`,
   `build/minify`) — target to replace in Phase 3.
3. **103 libraries vendored into `js/`** (multiple versions of fabric, jquery, flot, etc.) —
   target to move into `package.json` deps in Phase 2.
4. **Browser smoke test is not automatable here.** `npm test` → `build/quicktest` launches Chrome +
   the helper interactively; no headless browser tooling installed. CI (Phase 1) should add
   Playwright to load `tests/smoke.html` headless and assert an image renders.

## Reproduce this baseline
```bash
./configure
make allinone                 # should print nothing, leave js9-allinone.js unchanged
for f in js9.js js9.min.js js9-allinone.js js9support.js js9plugins.js js9Helper.js js9worker.js astroem.js; do
  node --check "$f" && echo "OK $f"
done
timeout 5 node js9Helper.js   # expect: "helper: 0.0.0.0 2718 ..." then still running (exit 124)
```
